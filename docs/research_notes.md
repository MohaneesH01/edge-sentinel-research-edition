# Research Notes

## Publication Goal

Working title:

**Edge Sentinel: A Low-Cost TinyML-Based Framework for Solar Fault Detection in Distributed Renewable Infrastructure**

## Paper Structure

1. Abstract
2. Introduction
3. Literature Review
4. System Architecture
5. Data Collection Methodology
6. Experimental Setup
7. Results
8. Discussion
9. Future Work
10. Conclusion

## Engineering Justification Checklist

Every feature should answer:

> What engineering problem does this solve?

Before adding a feature, record whether it improves:

- Detection latency
- Measurement reliability
- Dataset quality
- Edge deployment feasibility
- Fault interpretation
- Research credibility
